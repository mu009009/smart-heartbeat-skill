# 💓 心跳回复模板 - 简洁版

## 模板1: 所有正常
```
📊 HEARTBEAT检查完成

✅ 系统状态：全部正常
📅 检查时间：{time}

系统暴君宣言：检查完成！一切正常！🔥
```

## 模板2: 有警告
```
📊 HEARTBEAT检查完成

✅ 主系统：正常
⚠️ 警告：{warning}
📅 检查时间：{time}

系统暴君宣言：检查完成！注意警告！🔥
```

## 模板3: 有任务进行中
```
📊 HEARTBEAT检查完成

✅ 主系统：正常
🔄 任务：{task}进行中
📅 检查时间：{time}

系统暴君宣言：检查完成！任务进行中！🔥
```

## 模板4: 多服务器状态
```
📊 HEARTBEAT检查完成

✅ 主服务器：全部正常
🔄 小物服务器：{status}
📅 检查时间：{time}

系统暴君宣言：检查完成！等待任务！🔥
```

## 使用规则：
1. **字数限制**：不超过200字
2. **避免重复**：只说一次，不出现"回答："后重复
3. **简洁明了**：只保留关键信息
4. **时间标注**：包含检查时间
5. **风格保持**：保留系统暴君宣言

## 示例（压缩后）：
```
📊 HEARTBEAT检查完成

✅ 主服务器：全部正常
🔄 小物服务器：安装中 (npx openclaw@2026.3.8)

系统暴君宣言：检查完成！等待安装！🔥
```
